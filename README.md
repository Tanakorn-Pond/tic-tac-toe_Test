# Turn-Based Tic-Tac-Toe

link: https://tic-tac-toe-test-frontend.onrender.com

Deploy ผ่าน Render

## ฟีเจอร์หลัก (Features)

- **Authentication System**: ระบบสมัครสมาชิกและเข้าสู่ระบบ (Login/Register)
- **Game Lobby**:
  - **Bot Mode**: โหมดฝึกซ้อมเล่นกับ Bot
- **Modern UI**: ออกแบบด้วย Tailwind CSS 

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Backend**: Node.js (REST API)
- **Database**: Relational Database (PostgreSQL / MySQL)

## วิธีการรันโปรเจกต์ (Local Development)

### Prerequisites
- Node.js
- npm

### 1. Setup Backend
```bash
cd backend
npm install
npm start
```

### 2. Setup Frontend
```bash
cd Frontend
npm install
npm run dev
```
